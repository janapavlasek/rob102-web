Tutorial: Programming in Julia

Prerequisites

This tutorial assumes you have followed the instructions for installing Julia.

Launching a Jupyter Notebook

We will use Jupyter notebooks to write code in Julia quickly and easily visualize outputs. To launch a Jupyter notebook which uses Julia, open a Julia interpreter and enter the following commands:

using IJulia
notebook()

A webpage should open on your default browser with the home page for the Jupyter notebook server you just launched. Navigate to a folder which contains a Jupyter notebook and click on it to open it in a new tab. Or, create a new Julia notebook with New > Julia 1.6.1. To stop the Jupyter notebook server, do Ctrl-C (on Mac, do Cmd-C).

If you want to keep using the Julia interpreter, you can run the notebook in the background instead with notebook(detached=true).

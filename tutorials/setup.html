Using the ROB 102 Path Planning Web App

In VSCode, open the rob102-path-planning folder as a project. Open a Bash terminal in the root of this directory from VSCode. Build the Docker container using the helper script:

.docker_build.sh

When the build completes, run the container as follows:

.docker_run.sh

Now you have a Docker container called rob102_planning running. You should see it in the Docker Explorer panel, with a green icon to indicate that it is running. Go to http://localhost:8000. You should see the app, as shown below.
